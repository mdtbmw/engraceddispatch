# ENGRACED DISPATCH - Premium Logistics & Dispatch Android Application

## Overview
**ENGRACED DISPATCH** is an enterprise-grade logistics and parcel dispatch management platform built with Kotlin and Jetpack Compose. It features a complete end-to-end ecosystem for Customers, Employee Fleet Drivers, and Dispatch Administrators.

## Core Architectural Modules
1. **Customer Portal**: Create shipments, live map tracking with animated driver markers, AI dispatch assistant, ratings and tips.
2. **Employee Fleet Driver App**: Real-time dispatch manifest, company bike/van vehicle assignment, proximity-based arrival triggers, secure OTP delivery verification, and payroll/tips tracking.
3. **Admin Control Center**: Complete fleet overview, smart AI assignment engine, bulk driver reassignment, live map audit logs, and system overrides.

## Production Readiness
- **Zero Gig Terminology**: Strictly configured for company-employed dispatch drivers with formal manifests and payroll.
- **Firebase Backend Integration**: Real-time Firestore synchronization for shipments, rider telemetry, and admin audit logs.
- **Material 3 Design System**: Dark luxury theme featuring Obsidian, Charcoal, and Gold accents.
